结构体

struct 定义 成员访问 内存对齐 嵌套结构 初级结束

🎯 学习目标

  • ✓ 理解结构体的作用：将相关数据组合成一个整体
  • ✓ 掌握 struct 的定义语法和聚合初始化（C++11）
  • ✓ 使用点运算符 . 和箭头运算符 -> 访问成员
  • ✓ 理解内存对齐（padding）对 sizeof 的影响
  • ✓ 了解 struct 与 class 的区别，掌握嵌套结构体

🤔 为什么需要结构体？

假设你要表示一个学生，有姓名、年龄、成绩三个属性。没有结构体时，你需要用三个独立变量，容易混乱：

没有 struct vs 有 struct 
// ❌ 没有结构体：多个独立变量，难以管理多个学生
string student1_name = "张三";
int student1_age = 20;
double student1_gpa = 3.8;

string student2_name = "李四";
int student2_age = 21;
double student2_gpa = 3.5;
// 有 100 个学生怎么办？混乱！

// ✅ 使用结构体：将相关数据打包在一起
struct Student {
    string name;
    int age;
    double gpa;
};

Student s1 = {"张三", 20, 3.8};
Student s2 = {"李四", 21, 3.5};
Student class_list[100];  // 轻松管理 100 个学生
💡

结构体的本质：将逻辑上相关的数据组合成一个用户自定义类型。定义好 Student 后，它就像内置类型一样，可以声明变量、传递给函数、存入数组。

🏗️ 结构体定义与成员函数

C++ 的 struct 不仅可以包含数据成员，还可以包含成员函数（方法），让数据和行为结合在一起：

带成员函数的 struct 
#include <iostream>
#include <string>
using namespace std;

struct Student {
    // 数据成员
    string name;
    int age;
    double gpa;

    // 成员函数（const 表示不修改成员）
    void print() const {
        cout << name << " (年龄:" << age
             << " GPA:" << gpa << ")" << endl;
    }

    bool isHonors() const {
        return gpa >= 3.5;  // GPA >= 3.5 为荣誉生
    }
};

int main() {
    // 聚合初始化（C++11，按成员声明顺序）
    Student s1 = {"张三", 20, 3.8};

    // C++20 指定初始化（更清晰）
    // Student s2 = {.name="李四", .age=21, .gpa=3.2};

    // 访问成员：点运算符 .
    cout << s1.name << endl;   // 张三
    s1.age = 21;               // 修改成员

    s1.print();
    cout << (s1.isHonors() ? "荣誉生" : "普通生") << endl;

    // 结构体赋值（按值拷贝所有成员）
    Student s2 = s1;
    s2.name = "李四";    // 修改 s2 不影响 s1
    return 0;
}

🔑 成员访问：. 与 ->

访问结构体成员有两种运算符，取决于你有的是对象还是指针：

. 运算符 vs -> 运算符 
Student s = {"王五", 22, 3.9};

// 点运算符：直接访问对象的成员
cout << s.name << endl;    // 王五
s.gpa = 4.0;

// 箭头运算符：通过指针访问成员
Student* ptr = &s;
cout << ptr->name << endl;  // 王五（等价于 (*ptr).name）
ptr->age = 23;              // 修改原对象

// (*ptr).name 和 ptr->name 完全等价
// 箭头运算符 -> 是一个语法糖，让代码更简洁

// 动态分配的结构体
Student* dynamic = new Student{"赵六", 19, 3.1};
dynamic->gpa = 3.3;
dynamic->print();
delete dynamic;
dynamic = nullptr;
情形 运算符 示例
对象（值类型） . s.name
指针 -> ptr->name（等价于 (*ptr).name）

📚 结构体数组

结构体可以存入数组或 std::vector，方便管理多个同类型对象：

结构体数组与 vector 
#include <vector>
#include <algorithm>
using namespace std;

// C 风格结构体数组
Student class_arr[3] = {
    {"张三", 20, 3.8},
    {"李四", 21, 3.2},
    {"王五", 19, 3.9}
};
for (int i = 0; i < 3; i++) {
    class_arr[i].print();
}

// std::vector（推荐，动态大小）
vector<Student> students = {
    {"赵六", 22, 3.5},
    {"钱七", 20, 2.8},
    {"孙八", 21, 3.7}
};

// 遍历（const 引用，避免拷贝）
for (const auto& s : students) {
    s.print();
}

// 按 GPA 排序
sort(students.begin(), students.end(),
     [](const Student& a, const Student& b) {
         return a.gpa > b.gpa;  // 降序
     });

cout << "最高 GPA: " << students[0].name << endl;

🧩 内存对齐（Memory Alignment）

CPU 访问内存时，对特定类型有对齐要求（如 int 要求地址是 4 的倍数）。编译器会在成员之间插入填充字节（padding）来满足对齐要求，导致 sizeof(struct) 可能大于各成员之和：

内存对齐示例 
struct Padded {
    char c;    // 1 字节，offset 0
    // 3 字节填充（让 i 对齐到 4 字节边界）
    int i;     // 4 字节，offset 4
    char d;    // 1 字节，offset 8
    // 3 字节填充（让结构体大小是 4 的倍数）
};  // sizeof = 12

struct Packed {
    char c;    // 1 字节，offset 0
    char d;    // 1 字节，offset 1
    // 2 字节填充
    int i;     // 4 字节，offset 4
};  // sizeof = 8（成员排列顺序影响大小！）

cout << sizeof(Padded) << endl;  // 12
cout << sizeof(Packed) << endl;  // 8

// 技巧：把大类型放前面，可以减少 padding
// 按 double > int > short > char 的顺序排列成员
📌

为什么需要对齐？：CPU 读取 4 字节 int 时，要求数据的地址是 4 的倍数。如果不对齐，某些 CPU 要读取两次然后拼接（效率低），甚至会抛出硬件异常。编译器插入 padding 是为了让每个成员都满足对齐要求，以最大化访问效率。

🏗️ 嵌套结构体

结构体的成员可以是另一个结构体，构成层次化的数据组织：

嵌套结构体 
struct Address {
    string city;
    string street;
    int zipCode;
};

struct Person {
    string name;
    int age;
    Address addr;   // 嵌套：Address 是 Person 的成员
};

Person p = {"张三", 30, {"北京", "长安街", 100000}};

// 链式访问：.addr.city
cout << p.name << " 住在 " << p.addr.city << endl;

// 通过指针访问嵌套成员
Person* ptr = &p;
cout << ptr->addr.street << endl;  // 长安街
ptr->addr.zipCode = 100001;         // 修改嵌套成员

// 也可以独立初始化
Address home = {"上海", "南京路", 200000};
Person p2 = {"李四", 25, home};

⚖️ struct vs class

在 C++ 中，struct 和 class 几乎相同，唯一的技术差异是默认访问权限：

特性 struct class
默认访问权限 public private
默认继承权限 public 继承 private 继承
使用惯例 简单数据聚合（POD） 封装行为的对象
其他功能 完全相同（构造函数、继承、模板...）
struct vs class 访问权限对比 
struct Point {
    double x;  // 默认 public：外部可直接访问
    double y;
};

class Circle {
    double radius;  // 默认 private：外部不可直接访问
public:
    Circle(double r) : radius(r) {}
    double area() const { return 3.14159 * radius * radius; }
};

Point p;
p.x = 3.0;    // ✓ struct 默认 public，可以访问

Circle c(5.0);
// c.radius = 3.0;  // ✗ 编译错误：radius 是 private
cout << c.area() << endl;  // ✓ 通过公有函数访问

// 惯例：
// struct → 数据优先，成员公有，无复杂逻辑（如 Point, Color, Config）
// class  → 封装优先，隐藏实现细节，提供公有接口

🏷️ using 类型别名

using 可以为类型创建别名，让代码更简洁。也可以配合匿名结构体使用：

using 类型别名 
// 为已有类型创建别名（C++11 推荐，替代旧式 typedef）
using Point = struct { double x; double y; };
using Color = struct { int r; int g; int b; };

Point origin = {0.0, 0.0};
Point pt = {3.0, 4.0};
Color red = {255, 0, 0};

// 等价于旧式 typedef（不推荐）
// typedef struct { double x; double y; } Point;

// 也可以给复杂类型起别名
using IntMatrix = int[10][10];
using StringVec = vector<string>;  // 见中级课程

// using 更灵活，支持模板别名（typedef 不支持）
template<typename T>
using Pair = struct { T first; T second; };

🧪 综合代码示例

Rectangle 结构体综合演示 
#include <iostream>
#include <string>
#include <vector>
using namespace std;

struct Rectangle {
    double width;
    double height;
    string color;

    double area() const { return width * height; }
    double perimeter() const { return 2 * (width + height); }

    void print() const {
        cout << color << " 矩形: " << width << "x" << height
             << " 面积=" << area() << " 周长=" << perimeter() << endl;
    }
};

int main() {
    Rectangle r1 = {5.0, 3.0, "红色"};
    Rectangle r2 = {4.0, 4.0, "蓝色"};
    r1.print();
    r2.print();

    // 结构体指针
    Rectangle* ptr = &r1;
    ptr->width = 6.0;
    cout << "修改后: " << ptr->width << "x" << ptr->height << endl;

    // 结构体数组（用 vector）
    vector<Rectangle> shapes = {
        {3.0, 2.0, "绿色"}, {5.0, 5.0, "黄色"}, {1.0, 8.0, "紫色"}
    };
    double totalArea = 0;
    for (const auto& s : shapes) totalArea += s.area();
    cout << "总面积: " << totalArea << endl;

    return 0;
}
