➕ 算术运算符

算术运算符用于执行数学计算。C++ 提供了完整的四则运算和取余运算：

运算符 名称 示例 结果
+ 加法 5 + 3 8
- 减法 10 - 4 6
* 乘法 3 * 4 12
/ 除法（注意整数截断！） 7 / 2 3（不是 3.5！）
% 取余（模运算） 7 % 3 1（7 = 2×3 + 1）
算术运算符示例 
#include <iostream>

int main() {
    int a{10}, b{3};

    std::cout << "a + b = " << a + b << "\n";   // 13
    std::cout << "a - b = " << a - b << "\n";   // 7
    std::cout << "a * b = " << a * b << "\n";   // 30
    std::cout << "a / b = " << a / b << "\n";   // 3（整数截断）
    std::cout << "a % b = " << a % b << "\n";   // 1

    // 浮点除法
    std::cout << "10.0 / 3 = " << 10.0 / 3 << "\n"; // 3.33333

    // 取余的实用技巧
    std::cout << "判断奇偶: " << (a % 2 == 0 ? "偶数" : "奇数") << "\n";
    std::cout << "循环索引: " << 15 % 7 << "\n"; // 1，用于循环队列

    return 0;
}
⚠️

除以零是未定义行为！整数除以 0 会导致程序崩溃（运行时错误），浮点除以 0 会得到 inf 或 nan。在做除法前要确保除数不为零。

⚖️ 关系运算符

关系运算符用于比较两个值，结果总是 bool 类型（true 或 false）：

关系运算符 
#include <iostream>

int main() {
    int x{5}, y{10};

    std::cout << (x == y) << "\n";  // 0 (false) — 相等
    std::cout << (x != y) << "\n";  // 1 (true)  — 不等
    std::cout << (x <  y) << "\n";  // 1 (true)  — 小于
    std::cout << (x >  y) << "\n";  // 0 (false) — 大于
    std::cout << (x <= y) << "\n";  // 1 (true)  — 小于等于
    std::cout << (x >= y) << "\n";  // 0 (false) — 大于等于

    // 配合 boolalpha 输出 true/false 而非 1/0
    std::cout << std::boolalpha;
    std::cout << (x == y) << "\n";  // false
    std::cout << (x <  y) << "\n";  // true

    return 0;
}
🚫

经典错误：= 和 == 混淆！ if (x = 5) 是赋值（将 5 赋给 x，结果永远为真），if (x == 5) 才是比较。防范技巧：写 if (5 == x)（常量在左），如果误写成 = 会报编译错误。

🧩 逻辑运算符与短路求值

逻辑运算符用于组合多个条件。C++ 的 && 和 || 具有短路求值（Short-circuit Evaluation）特性：

逻辑运算符 
#include <iostream>

int main() {
    bool a{true}, b{false};

    // && (AND)：两者都为 true 才返回 true
    std::cout << (a && b) << "\n";  // false
    std::cout << (a && a) << "\n";  // true

    // || (OR)：至少一个为 true 就返回 true
    std::cout << (a || b) << "\n";  // true
    std::cout << (b || b) << "\n";  // false

    // ! (NOT)：取反
    std::cout << (!a) << "\n";  // false
    std::cout << (!b) << "\n";  // true

    // ===== 短路求值 =====
    int x{0};

    // && 短路：左边为 false，右边不执行
    // 如果 x == 0，则 (1/x) 不会被计算，避免除零崩溃
    if (x != 0 && (10 / x > 1)) {
        std::cout << "安全计算\n";
    } else {
        std::cout << "x 为 0，短路保护生效\n";
    }

    // || 短路：左边为 true，右边不执行
    int* ptr = nullptr;
    if (ptr == nullptr || *ptr == 0) {
        std::cout << "空指针检查通过，不会解引用\n";
    }

    return 0;
}
💡

短路求值的实用价值：检查指针或除数之前，先用 && 的短路特性进行安全性检查。如 ptr != nullptr && ptr->value > 0，只有指针非空时才访问其成员。

📝 赋值运算符

复合赋值运算符是"先运算后赋值"的简写形式，代码更简洁：

赋值运算符 
#include <iostream>

int main() {
    int x{10};

    x += 5;   // 等同于 x = x + 5;  → 15
    std::cout << "x += 5:  " << x << "\n";

    x -= 3;   // 等同于 x = x - 3;  → 12
    std::cout << "x -= 3:  " << x << "\n";

    x *= 2;   // 等同于 x = x * 2;  → 24
    std::cout << "x *= 2:  " << x << "\n";

    x /= 4;   // 等同于 x = x / 4;  → 6
    std::cout << "x /= 4:  " << x << "\n";

    x %= 4;   // 等同于 x = x % 4;  → 2
    std::cout << "x %= 4:  " << x << "\n";

    // 链式赋值（右结合）
    int a, b, c;
    a = b = c = 100;   // c = 100, b = 100, a = 100
    std::cout << a << " " << b << " " << c << "\n";

    return 0;
}

🔢 自增与自减运算符

++ 和 -- 有前置和后置两种形式，区别在于返回值的时机：

前置 vs 后置 
#include <iostream>

int main() {
    int a{5};

    // 前置 ++：先自增，再使用（返回自增后的值）
    int b = ++a;   // a 先变 6，b = 6
    std::cout << "++a: a=" << a << " b=" << b << "\n";  // a=6, b=6

    a = 5;  // 重置

    // 后置 ++：先使用，再自增（返回自增前的值）
    int c = a++;   // c = 5，然后 a 变 6
    std::cout << "a++: a=" << a << " c=" << c << "\n";  // a=6, c=5

    // 实际用法：在循环中通常两者效果一样
    for (int i = 0; i < 5; ++i) {  // 推荐使用前置 ++
        std::cout << i << " ";
    }
    std::cout << "\n";

    return 0;
}
🚀

循环中推荐使用前置 ++i：对于内置类型，两者性能相同。但对于自定义类型（如迭代器），后置 i++ 需要创建一个临时对象保存旧值，效率略低。养成使用 ++i 的习惯是好实践。

🔌 位运算符

位运算符直接操作整数的二进制位，在底层编程、标志位操作、性能优化等场景非常有用：

位运算符详解 
#include <iostream>
#include <bitset>   // 用于二进制打印

int main() {
    unsigned int a{0b00001100};  // 12，二进制 0000 1100
    unsigned int b{0b00001010};  // 10，二进制 0000 1010

    // & (AND)：对应位都为 1 才为 1
    std::cout << "a & b  = " << std::bitset<8>(a & b)  << " (" << (a & b)  << ")\n"; // 0000 1000 = 8

    // | (OR)：对应位有一个为 1 就为 1
    std::cout << "a | b  = " << std::bitset<8>(a | b)  << " (" << (a | b)  << ")\n"; // 0000 1110 = 14

    // ^ (XOR)：对应位不同为 1，相同为 0
    std::cout << "a ^ b  = " << std::bitset<8>(a ^ b)  << " (" << (a ^ b)  << ")\n"; // 0000 0110 = 6

    // ~ (NOT)：按位取反
    std::cout << "~a     = " << std::bitset<8>(~a & 0xFF) << "\n";  // 1111 0011

    // << (左移)：相当于乘以 2 的幂
    std::cout << "a << 1 = " << std::bitset<8>(a << 1) << " (" << (a << 1) << ")\n"; // 24 = 12*2

    // >> (右移)：相当于除以 2 的幂
    std::cout << "a >> 1 = " << std::bitset<8>(a >> 1) << " (" << (a >> 1) << ")\n"; // 6 = 12/2

    // 实际应用：标志位操作
    const unsigned int READ    = 0b001;  // 1
    const unsigned int WRITE   = 0b010;  // 2
    const unsigned int EXECUTE = 0b100;  // 4

    unsigned int perms = READ | WRITE;   // 设置读和写权限
    std::cout << "\n权限设置: " << std::bitset<3>(perms) << "\n";
    std::cout << "有读权限: " << std::boolalpha << (bool)(perms & READ)    << "\n";
    std::cout << "有执行权限: " << std::boolalpha << (bool)(perms & EXECUTE) << "\n";

    perms |= EXECUTE;  // 添加执行权限
    perms &= ~WRITE;   // 移除写权限
    std::cout << "修改后权限: " << std::bitset<3>(perms) << "\n";

    return 0;
}

📊 运算符优先级

当表达式中出现多个运算符时，优先级决定了计算顺序。优先级高的先计算，相同优先级的按结合性计算。

优先级 运算符 描述 结合性
1（最高） () [] . -> 括号、下标、成员访问 左→右
2 ++ -- ! ~ (type) * & sizeof 一元运算符 右→左
3 * / % 乘、除、取余 左→右
4 + - 加法、减法 左→右
5 << >> 位移 左→右
6 < <= > >= 关系比较 左→右
7 == != 相等比较 左→右
8 & 按位与 左→右
9 ^ 按位异或 左→右
10 | 按位或 左→右
11 ⚠️ && 逻辑与（短路） 左→右
12 ⚠️ || 逻辑或（短路） 左→右
13 ?: 条件（三目） 右→左
14 = += -= *= /= %= &= |= ^= <<= >>= 赋值 右→左
15（最低） , 逗号 左→右
⚠️

常见陷阱：&（位与，优先级8）比 ==（优先级7）低！所以 x & 0xFF == 0 实际上是 x & (0xFF == 0)，即 x & false，结果是 0。正确写法：(x & 0xFF) == 0。遇到不确定的情况，加括号！

优先级陷阱示例 
#include <iostream>

int main() {
    int x{6};

    // 陷阱1：& 优先级低于 ==
    // std::cout << (x & 1 == 0) << "\n";  // 实际是 x & (1 == 0) = x & false = 0
    std::cout << ((x & 1) == 0) << "\n";   // 正确：6 的末位是 0，所以是偶数 → true

    // 陷阱2：！的结合性（右→左）
    int a{1}, b{2};
    // !a + b 是 (!a) + b = 0 + 2 = 2，不是 !(a + b)
    std::cout << (!a + b) << "\n";  // 2，而非 !(1+2)=false

    // 正确做法：有疑虑时加括号
    std::cout << (2 + 3 * 4) << "\n";   // 14 (* 优先于 +)
    std::cout << ((2 + 3) * 4) << "\n"; // 20 (括号改变优先级)

    return 0;
}

❓ 条件（三目）运算符

三目运算符 ?: 是 if-else 的简洁替代，适合在表达式中嵌入简单的条件选择：

三目运算符 
#include <iostream>

int main() {
    int score{85};

    // 语法：条件 ? 真时的值 : 假时的值
    std::string grade = (score >= 60) ? "及格" : "不及格";
    std::cout << "成绩: " << grade << "\n";  // 及格

    // 求两数最大值
    int a{7}, b{12};
    int maxVal = (a > b) ? a : b;
    std::cout << "最大值: " << maxVal << "\n";  // 12

    // 嵌套三目（可读性差，不推荐）
    int x{75};
    std::string level = (x >= 90) ? "优秀" : (x >= 75) ? "良好" : (x >= 60) ? "及格" : "不及格";
    std::cout << "评级: " << level << "\n";  // 良好

    // 在输出中使用（常见用法）
    for (int i = 1; i <= 5; ++i) {
        std::cout << i << (i % 2 == 0 ? "（偶）" : "（奇）") << "\n";
    }

    return 0;
}

🧪 动手练习

综合练习各类运算符：

综合运算符练习 
#include <iostream>

int main() {
    int a{15}, b{4};

    // 算术
    std::cout << "=== 算术运算 ===\n";
    std::cout << a << " / " << b << " = " << a / b << "\n";
    std::cout << a << " % " << b << " = " << a % b << "\n";

    // 关系和逻辑
    std::cout << "\n=== 关系与逻辑 ===\n";
    bool cond = (a > 10) && (b < 10);
    std::cout << std::boolalpha << "a>10 && b<10: " << cond << "\n";

    // 位运算
    std::cout << "\n=== 位运算 ===\n";
    std::cout << "a << 1 = " << (a << 1) << "  (×2)\n";
    std::cout << "a >> 1 = " << (a >> 1) << "  (÷2 整数)\n";
    std::cout << "a & b  = " << (a & b)  << "\n";

    // 三目
    std::cout << "\n=== 三目运算 ===\n";
    std::cout << "最大值: " << (a > b ? a : b) << "\n";
    std::cout << (a % 2 == 0 ? "a是偶数" : "a是奇数") << "\n";

    return 0;
}

📋 课后作业

理解题

解释 2 + 3 * 4 - 1 的计算顺序和最终结果。然后用括号改写，使结果变为 15。

编程题

编写程序：输入一个整数，判断并输出：① 是奇数还是偶数（用 %）；② 是否能被 3 和 5 同时整除（用 &&）；③ 用位运算检查最低位是 0 还是 1（用 &）。

挑战题

仅用位运算（不用 +），实现两个整数的加法（提示：用异或 ^ 计算无进位和，用 & 和左移计算进位，循环直到进位为 0）。
