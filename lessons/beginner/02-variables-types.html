变量与基本类型

变量 数据类型 内存

🎯 学习目标

  • ✓ 掌握 C++ 基本数据类型（int, float, double, char, bool）及其大小
  • ✓ 理解变量的声明、初始化、赋值与作用域
  • ✓ 了解 sizeof 运算符、类型转换和整数溢出

📦 变量是什么

变量是内存中一块有名字的存储空间。你可以把它想象成一个标注了名称的盒子，用来存放数据。程序运行时，每个变量都对应一个内存地址，存储特定类型的值。

不同类型的变量在内存中占用不同大小的空间

🔢 基本数据类型

C++ 提供了一系列内置的基本数据类型。了解每种类型的大小和取值范围非常重要，可以帮助你选择合适的类型并避免错误。

（64位系统上的典型大小，实际大小可能因平台不同而异）

类型 大小 取值范围 用途
bool 1 字节 true / false 逻辑值、条件标志
char 1 字节 -128 ~ 127（或 0 ~ 255） 单个字符（ASCII）
short 2 字节 -32,768 ~ 32,767 小整数，内存敏感场景
int 4 字节 约 -21亿 ~ 21亿 通用整数（最常用）
long / long long 8 字节 约 ±922京 大整数（文件大小、时间戳）
float 4 字节 ±3.4×10³⁸（约7位精度） 单精度浮点（图形学常用）
double 8 字节 ±1.7×10³⁰⁸（约15位精度） 双精度浮点（默认浮点类型）

✍️ 变量声明与初始化

C++ 变量必须先声明再使用。声明指定了变量的类型和名称，初始化则赋予了初始值。

变量声明的各种形式 
#include <iostream>

int main() {
    // 1. 声明但不初始化（值是未定义的垃圾值！）
    int a;          // 危险！a 的值不确定

    // 2. 声明并赋初值（C 风格）
    int b = 10;     // 赋值初始化
    double pi = 3.14159;

    // 3. C++11 统一初始化语法（推荐）
    int c{42};      // 大括号初始化，更安全
    double x{2.71828};
    bool flag{true};
    char grade{'A'};

    // 4. 同时声明多个同类型变量
    int m = 1, n = 2, k = 3;

    // 5. auto 类型推导（C++11）
    auto result = 100;      // 编译器推导为 int
    auto price = 9.99;      // 推导为 double

    std::cout << "b = " << b << std::endl;
    std::cout << "pi = " << pi << std::endl;
    std::cout << "grade = " << grade << std::endl;
    std::cout << "flag = " << flag << std::endl;  // 输出 1（true）

    return 0;
}
🚫

未初始化变量是 C++ 中最常见的 bug 来源之一。未初始化的局部变量包含"垃圾值"（上次使用那块内存残留的数据）。始终在声明时初始化变量！

💡

大括号初始化（int x{42};）的优势：它会在发生精度缩窄时产生编译错误，例如 int x{3.14}; 会报错，而 int x = 3.14; 则会静默截断为 3。

🔒 const 常量

const 关键字将变量标记为只读，声明后值不能被修改。使用常量可以避免意外修改，让代码更安全、更具可读性。

const 的使用 
#include <iostream>

int main() {
    // const 变量必须在声明时初始化
    const double PI = 3.14159265358979;
    const int MAX_STUDENTS = 100;
    const char NEWLINE = '\n';

    // 错误！不能修改 const 变量
    // PI = 3.14;  // 编译错误：assignment of read-only variable

    // constexpr：编译期常量（C++11，比 const 更严格，性能更好）
    constexpr int ARRAY_SIZE = 1000;
    constexpr double GRAVITY = 9.81;

    std::cout << "圆周率 PI = " << PI << NEWLINE;
    std::cout << "最大学生数 = " << MAX_STUDENTS << NEWLINE;
    std::cout << "重力加速度 = " << GRAVITY << " m/s²" << NEWLINE;

    return 0;
}

📏 sizeof 运算符

sizeof 是一个编译期运算符，返回类型或变量占用的字节数。它不是函数，不需要函数调用的开销。

sizeof 演示 
#include <iostream>

int main() {
    // 对类型使用 sizeof
    std::cout << "sizeof(bool)        = " << sizeof(bool)        << " 字节\n";
    std::cout << "sizeof(char)        = " << sizeof(char)        << " 字节\n";
    std::cout << "sizeof(short)       = " << sizeof(short)       << " 字节\n";
    std::cout << "sizeof(int)         = " << sizeof(int)         << " 字节\n";
    std::cout << "sizeof(long)        = " << sizeof(long)        << " 字节\n";
    std::cout << "sizeof(long long)   = " << sizeof(long long)   << " 字节\n";
    std::cout << "sizeof(float)       = " << sizeof(float)       << " 字节\n";
    std::cout << "sizeof(double)      = " << sizeof(double)      << " 字节\n";

    // 对变量使用 sizeof
    int x = 42;
    double y = 3.14;
    std::cout << "sizeof(x)           = " << sizeof(x) << " 字节\n";
    std::cout << "sizeof(y)           = " << sizeof(y) << " 字节\n";

    return 0;
}

/* 典型 64 位系统输出：
sizeof(bool)        = 1 字节
sizeof(char)        = 1 字节
sizeof(short)       = 2 字节
sizeof(int)         = 4 字节
sizeof(long)        = 8 字节
sizeof(long long)   = 8 字节
sizeof(float)       = 4 字节
sizeof(double)      = 8 字节 */

🔄 类型转换

C++ 中类型转换分为隐式转换（编译器自动执行）和显式转换（程序员手动指定）。

类型转换示例 
#include <iostream>

int main() {
    // ===== 隐式转换（自动提升到更大的类型）=====
    int a = 10;
    double b = a;          // int → double，无损失
    std::cout << "b = " << b << "\n";  // 10.0

    double c = 3.99;
    int d = c;             // double → int，截断小数！
    std::cout << "d = " << d << "\n";  // 3（不是4！）

    // ===== 显式转换（static_cast，推荐方式）=====
    double price = 9.75;
    int floorPrice = static_cast<int>(price);  // 明确意图
    std::cout << "floorPrice = " << floorPrice << "\n";  // 9

    // 整数除法 vs 浮点除法
    int x = 7, y = 2;
    std::cout << "7 / 2 = " << x / y << "\n";            // 3（整数除法！）
    std::cout << "7.0 / 2 = " << 7.0 / 2 << "\n";        // 3.5
    std::cout << "7 / 2.0 = " << x / 2.0 << "\n";        // 3.5
    std::cout << "(double)7/2 = " << (double)x / y << "\n"; // 3.5，C 风格转换

    return 0;
}
⚠️

初学者最常见的陷阱：整数除法截断。计算 7 / 2 结果是 3 而不是 3.5！只要两个操作数都是整数，C++ 就执行整数除法。要得到浮点结果，至少一个操作数必须是浮点类型。

💥 整数溢出

当一个整数的值超出其类型能表示的最大范围时，就会发生溢出（Overflow）。有符号整数溢出是未定义行为（UB），无符号整数溢出会"回绕"。

整数溢出演示 
#include <iostream>
#include <climits>    // 包含各类型的极限值

int main() {
    // 查看 int 的边界值
    std::cout << "int 最大值: " << INT_MAX << "\n";  // 2147483647
    std::cout << "int 最小值: " << INT_MIN << "\n";  // -2147483648

    // 演示有符号整数溢出（行为未定义，但通常表现为回绕）
    int maxVal = INT_MAX;       // 2147483647
    int overflow = maxVal + 1;  // 未定义行为！通常变成 -2147483648
    std::cout << "INT_MAX + 1 = " << overflow << "\n";

    // 无符号整数溢出：安全地回绕（定义好的行为）
    unsigned int u = 0;
    unsigned int underflow = u - 1;   // 回绕到 UINT_MAX（4294967295）
    std::cout << "0u - 1 = " << underflow << "\n";

    // 用 long long 避免溢出
    long long big = 3000000000LL;   // 后缀 LL 表示 long long
    std::cout << "big = " << big << "\n";

    return 0;
}
🚫

有符号整数溢出是未定义行为（Undefined Behavior）！编译器有权假设它不会发生，并据此进行优化，可能导致完全出乎意料的结果。安全写法：在运算前检查值的范围，或使用更大的类型（如 long long）。

🧪 动手练习

综合演示各种变量类型、sizeof 和类型转换：

综合练习（可复制到本地运行） 
#include <iostream>

int main() {
    // 声明各种类型变量
    bool isStudent{true};
    char initial{'Z'};
    int age{20};
    double gpa{3.85};

    // 输出变量值和大小
    std::cout << "isStudent = " << isStudent << " (size: " << sizeof(isStudent) << "B)\n";
    std::cout << "initial   = " << initial   << " (size: " << sizeof(initial)   << "B)\n";
    std::cout << "age       = " << age       << " (size: " << sizeof(age)       << "B)\n";
    std::cout << "gpa       = " << gpa       << " (size: " << sizeof(gpa)       << "B)\n";

    // 整数除法 vs 浮点除法
    std::cout << "\n7 / 2 = " << 7 / 2 << " (整数除法)\n";
    std::cout << "7.0 / 2 = " << 7.0 / 2 << " (浮点除法)\n";

    return 0;
}
