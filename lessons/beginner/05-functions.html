🤔 为什么需要函数？

想象你在编写一个程序，需要在多个地方计算圆的面积。没有函数时，你需要重复写同样的计算代码。有了函数，你只需写一次，然后到处调用：

优点 说明
代码复用 一处定义，多处调用，避免重复代码
模块化 将大问题分解为小问题，每个函数负责一件事
可读性 函数名即注释，calculateCircleArea(r) 一目了然
易于测试 可以单独测试每个函数，找 bug 更容易

🏗️ 函数的基本结构

C++ 函数由四个部分组成：返回类型、函数名、参数列表和函数体：

函数基本结构 
// 返回类型  函数名   参数列表
   int      add    (int a, int b) {
    // 函数体
    return a + b;  // 返回值
}

// void 表示函数不返回任何值
void greet(string name) {
    cout << "你好，" << name << "！" << endl;
    // 不需要 return（可以有 return; 来提前退出）
}

// 调用函数
int result = add(3, 5);   // result = 8
greet("世界");             // 输出：你好，世界！
💡

函数命名建议：使用动词或动词短语，如 calculateArea、printResult、isValid。对于返回布尔值的函数，通常以 is、has、can 开头，使代码更像英语句子。

📋 声明与定义分离

C++ 编译器从上到下读取代码。如果在 main 中调用一个尚未"见过"的函数，编译器会报错。解决方案是使用函数原型（声明）：

函数原型 vs 定义 
#include <iostream>
using namespace std;

// ===== 函数声明（原型）=====
// 只告诉编译器：函数的名字、参数类型、返回类型
// 参数名可以省略，但写上更清晰
int add(int a, int b);       // 声明
void printBanner(string msg); // 声明

int main() {
    // 因为上面有了声明，可以在这里调用
    cout << add(3, 5) << endl;  // 输出 8
    printBanner("C++ 学习");
    return 0;
}

// ===== 函数定义（实现）=====
// 可以放在 main 之后
int add(int a, int b) {
    return a + b;
}

void printBanner(string msg) {
    cout << "====== " << msg << " ======" << endl;
}

// 在实际项目中：
// 声明 → 放在 .h 头文件中
// 定义 → 放在 .cpp 源文件中
📌

头文件的作用：#include <iostream> 就是在引入一个头文件，它包含了 cout、cin 等函数/对象的声明。实际的实现（定义）在标准库的二进制文件中，链接时才会合并。

📦 参数传递——按值传递

C++ 默认使用按值传递（pass by value）：函数接收的是参数的副本，在函数内修改参数，不会影响调用者的原始变量：

按值传递演示 
#include <iostream>
using namespace std;

void tryToChange(int x) {
    x = 999;   // 只修改了副本！
    cout << "函数内 x = " << x << endl;  // 999
}

void doubleValue(int n) {
    n = n * 2;
    cout << "函数内 n = " << n << endl;
}

int main() {
    int a = 42;
    cout << "调用前 a = " << a << endl;  // 42

    tryToChange(a);   // 传递 a 的副本

    cout << "调用后 a = " << a << endl;  // 仍然是 42！

    int num = 10;
    doubleValue(num);
    cout << "调用后 num = " << num << endl;  // 仍然是 10

    return 0;
}
// 输出：
// 调用前 a = 42
// 函数内 x = 999
// 调用后 a = 42   ← 没有改变！
💡

预告：如果想让函数修改调用者的变量，需要使用按引用传递（int& x）或按指针传递（int* x）。这两种方式将在第 7 课《指针与引用》中详细讲解。

↩️ 函数返回值

return 语句将值传回调用者，并立即结束函数执行。一个函数可以有多个 return 语句：

返回值详解 
#include <iostream>
using namespace std;

// 多个 return 点（early return 风格）
string grade(int score) {
    if (score >= 90) return "优秀";
    if (score >= 80) return "良好";
    if (score >= 70) return "中等";
    if (score >= 60) return "及格";
    return "不及格";  // 所有情况都要有 return
}

// void 函数：不返回值，但可以提前退出
void printPositive(int n) {
    if (n <= 0) {
        cout << "不是正数，跳过" << endl;
        return;  // 提前退出，不执行后面的代码
    }
    cout << "正数：" << n << endl;
}

// 返回值可以用于进一步计算
int square(int n) { return n * n; }
int sumOfSquares(int a, int b) {
    return square(a) + square(b);  // 使用函数返回值
}

int main() {
    cout << grade(95) << endl;    // 优秀
    cout << grade(73) << endl;    // 中等
    printPositive(-5);              // 不是正数，跳过
    printPositive(8);               // 正数：8
    cout << sumOfSquares(3, 4) << endl;  // 25
    return 0;
}

🔀 函数重载

C++ 允许多个函数使用相同的名字，但参数类型或数量不同。编译器根据调用时传入的参数自动选择合适的版本：

函数重载 
#include <iostream>
using namespace std;

// 同名函数，不同参数类型
int add(int a, int b) {
    cout << "[int版本] ";
    return a + b;
}

double add(double a, double b) {
    cout << "[double版本] ";
    return a + b;
}

// 同名函数，不同参数数量
int add(int a, int b, int c) {
    cout << "[三参数版本] ";
    return a + b + c;
}

// 打印不同类型（重载的经典用例）
void print(int n)    { cout << "整数: " << n << endl; }
void print(double d) { cout << "浮点: " << d << endl; }
void print(string s) { cout << "字符串: " << s << endl; }

int main() {
    cout << add(1, 2) << endl;          // [int版本] 3
    cout << add(1.5, 2.5) << endl;      // [double版本] 4.0
    cout << add(1, 2, 3) << endl;       // [三参数版本] 6

    print(42);        // 整数: 42
    print(3.14);      // 浮点: 3.14
    print("Hello");   // 字符串: Hello

    return 0;
}
// 注意：重载只看参数，不看返回类型！
// int foo();  和  double foo();  同时存在会导致编译错误
🚫

重载的判断依据：参数的类型和数量（统称为函数签名）。返回类型不参与重载判断。如果两个函数只有返回类型不同，编译器无法区分，会产生错误。

⚙️ 默认参数

可以为参数提供默认值，调用时如果不传该参数，则使用默认值。默认参数必须从右往左设置：

默认参数 
#include <iostream>
using namespace std;

// 默认参数必须从右往左设置
void printLine(char c = '-', int n = 40) {
    for (int i = 0; i < n; ++i) cout << c;
    cout << endl;
}

// 混合：部分有默认值（默认值参数必须在右边）
string createTag(string content,
                 string tag = "p",
                 string cls = "") {
    string result = "<" + tag;
    if (!cls.empty()) result += " class=\"" + cls + "\"";
    result += ">" + content + "</" + tag + ">";
    return result;
}

int main() {
    printLine();              // 使用全部默认值：40个'-'
    printLine('=');           // 覆盖第一个：40个'='
    printLine('*', 20);       // 覆盖全部：20个'*'

    cout << createTag("Hello") << endl;
    // 输出：<p>Hello</p>

    cout << createTag("Title", "h1") << endl;
    // 输出：<h1>Title</h1>

    cout << createTag("Info", "div", "info-box") << endl;
    // 输出：<div class="info-box">Info</div>

    return 0;
}

// 错误示例：
// void bad(int a = 0, int b);  // 错误！有默认值的参数必须在最右边

🌀 递归函数

递归是指函数调用自身。每个递归函数必须有一个基准情况（base case）来终止递归，否则会无限调用导致栈溢出：

递归：阶乘与斐波那契 
#include <iostream>
using namespace std;

// 阶乘：n! = n × (n-1)!，且 0! = 1
int factorial(int n) {
    // 基准情况（Base Case）：终止递归
    if (n <= 1) return 1;

    // 递归情况：问题规模缩小
    return n * factorial(n - 1);
}
// 执行过程：factorial(4)
//   = 4 * factorial(3)
//   = 4 * 3 * factorial(2)
//   = 4 * 3 * 2 * factorial(1)
//   = 4 * 3 * 2 * 1  = 24

// 斐波那契：fib(n) = fib(n-1) + fib(n-2)
int fibonacci(int n) {
    if (n <= 1) return n;  // 基准情况：fib(0)=0, fib(1)=1
    return fibonacci(n-1) + fibonacci(n-2);
}
// 注意：这个实现有大量重复计算，效率低下
// 生产代码应使用动态规划或迭代

int main() {
    for (int i = 0; i <= 7; ++i)
        cout << i << "! = " << factorial(i) << endl;

    cout << "\n斐波那契数列: ";
    for (int i = 0; i < 10; ++i)
        cout << fibonacci(i) << " ";
    cout << endl;
    return 0;
}

factorial(3) 的调用栈

每次调用会在栈上创建新的栈帧，返回后栈帧弹出。如果递归太深，栈空间耗尽就会发生栈溢出（Stack Overflow）。

⚠️

递归的优缺点：优点是代码简洁优雅，适合描述天然递归的问题（树遍历、分治算法）。缺点是每次调用需要保存栈帧，内存消耗大；递归层数过深会导致栈溢出（Stack Overflow）；很多递归（如朴素斐波那契）有大量重复计算，效率低。

🧪 综合代码示例

函数综合演示 
#include <iostream>
using namespace std;

// 函数声明（原型）
int factorial(int n);
int fibonacci(int n);
double average(double a, double b, double c = 0.0);

int main() {
    // 函数调用演示
    cout << "5! = " << factorial(5) << endl;        // 120
    cout << "fib(10) = " << fibonacci(10) << endl;  // 55

    // 默认参数：只传两个参数时，c 默认为 0.0
    cout << "average(3, 7) = " << average(3, 7) << endl;      // 3.333
    cout << "average(1, 2, 3) = " << average(1, 2, 3) << endl; // 2.0
    return 0;
}

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);  // 递归调用
}

int fibonacci(int n) {
    if (n <= 1) return n;
    return fibonacci(n-1) + fibonacci(n-2);
}

double average(double a, double b, double c) {
    return (a + b + c) / 3.0;
}
