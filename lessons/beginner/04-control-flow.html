🔀 if / else if / else 条件语句

条件语句让程序根据不同情况执行不同的代码分支。这是程序"做决定"的基本机制：

if / else if / else 结构 
#include <iostream>

int main() {
    int score{82};

    // 基本 if-else if-else 链
    if (score >= 90) {
        std::cout << "优秀\n";
    } else if (score >= 80) {
        std::cout << "良好\n";       // 输出：良好
    } else if (score >= 70) {
        std::cout << "中等\n";
    } else if (score >= 60) {
        std::cout << "及格\n";
    } else {
        std::cout << "不及格\n";
    }

    // 嵌套 if
    int age{20};
    bool hasLicense{true};
    if (age >= 18) {
        if (hasLicense) {
            std::cout << "可以开车\n";
        } else {
            std::cout << "需要驾照\n";
        }
    } else {
        std::cout << "年龄不足\n";
    }

    return 0;
}
🚫

经典 Bug：忘记大括号！

当 if 后只有一条语句时，大括号是可选的，但这是一个危险的习惯：

大括号陷阱 
// 危险！看起来像 if 块，实际上只有第一条语句受 if 控制
if (x > 0)
    std::cout << "正数\n";
    std::cout << "这行总是执行！\n";  // 误导性缩进

// 正确写法：始终使用大括号
if (x > 0) {
    std::cout << "正数\n";
    std::cout << "这行也在 if 块中\n";
}

// C++17：if 初始化语句
if (int val = compute(); val > 0) {  // val 只在 if 块中可见
    std::cout << "正数: " << val << "\n";
}
💡

最佳实践：无论 if 块中只有多少条语句，始终使用大括号。这可以防止日后添加代码时引入 bug，也是主流 C++ 编码规范的要求。

🔧 switch 语句

switch 语句适合于对同一变量进行多个离散值的比较，比多个 if-else if 更清晰、有时也更高效：

switch 基本用法 
#include <iostream>

int main() {
    int day{3};  // 1=周一, ..., 7=周日

    switch (day) {
        case 1:
            std::cout << "周一\n";
            break;  // 必须的！否则会 fall-through 到下一个 case
        case 2:
            std::cout << "周二\n";
            break;
        case 3:
            std::cout << "周三\n";
            break;
        case 4:
            std::cout << "周四\n";
            break;
        case 5:
            std::cout << "周五\n";
            break;
        case 6:
        case 7:
            // case 6 fall-through 到 case 7，共享代码（有意为之）
            std::cout << "周末！\n";
            break;
        default:
            std::cout << "无效的天数\n";
            break;
    }

    // 使用 [[fallthrough]] 明确标注有意的 fall-through（C++17）
    char grade{'B'};
    switch (grade) {
        case 'A':
            std::cout << "获得奖学金\n";
            [[fallthrough]];  // 明确告诉编译器这里有意 fall-through
        case 'B':
        case 'C':
            std::cout << "成绩合格\n";
            break;
        case 'D':
        case 'F':
            std::cout << "需要补考\n";
            break;
        default:
            std::cout << "未知等级\n";
    }

    return 0;
}
⚠️

Fall-through 是最常见的 switch Bug！忘记写 break，程序会继续执行下一个 case 的代码，直到遇到 break 或 switch 结束。如果 fall-through 是有意的，用 [[fallthrough]]; 注释说明，避免编译器警告。switch 只支持整数类型（int、char、enum），不支持字符串或浮点数。

🔄 for 循环

for 循环是最常用的循环形式，适合已知循环次数的场景：

for 循环详解 
#include <iostream>

int main() {
    // 语法：for (初始化; 条件; 更新) { 循环体 }
    //       初始化：循环开始前执行一次
    //       条件：每次循环前检查，false 时退出
    //       更新：每次循环体执行完后执行

    // 基本 for 循环：从 1 到 5
    for (int i = 1; i <= 5; ++i) {
        std::cout << i << " ";
    }
    std::cout << "\n";  // 1 2 3 4 5

    // 递减循环
    for (int i = 5; i >= 1; --i) {
        std::cout << i << " ";
    }
    std::cout << "\n";  // 5 4 3 2 1

    // 步长为 2
    for (int i = 0; i <= 10; i += 2) {
        std::cout << i << " ";
    }
    std::cout << "\n";  // 0 2 4 6 8 10

    // C++11 范围 for 循环（用于容器/数组）
    int arr[] = {10, 20, 30, 40, 50};
    for (int val : arr) {
        std::cout << val << " ";
    }
    std::cout << "\n";  // 10 20 30 40 50

    // 用 auto 简化范围 for
    for (auto val : arr) {
        std::cout << val * 2 << " ";
    }
    std::cout << "\n";  // 20 40 60 80 100

    // 三段式可以省略（但分号不能省）
    int sum = 0, i = 1;
    for (; i <= 100; ++i) {   // 省略初始化
        sum += i;
    }
    std::cout << "1到100的和 = " << sum << "\n";  // 5050

    return 0;
}

🔁 while 循环

while 循环适合不确定循环次数、只知道终止条件的场景：

while 循环 
#include <iostream>

int main() {
    // 语法：while (条件) { 循环体 }
    // 条件为 true 时执行，可能一次都不执行

    int count{1};
    while (count <= 5) {
        std::cout << count << " ";
        ++count;  // 别忘了更新！否则死循环
    }
    std::cout << "\n";  // 1 2 3 4 5

    // 实用案例：数字猜谜（模拟）
    int secret{42};
    int guess{0};
    int attempts{0};

    // 模拟输入（实际程序中用 cin）
    int simGuesses[] = {10, 60, 42};  // 模拟的猜测序列
    int guessIdx{0};

    while (guess != secret) {
        guess = simGuesses[guessIdx++];
        ++attempts;

        if (guess < secret) {
            std::cout << "猜测 " << guess << "：太小了！\n";
        } else if (guess > secret) {
            std::cout << "猜测 " << guess << "：太大了！\n";
        } else {
            std::cout << "猜对了！答案是 " << secret
                      << "，用了 " << attempts << " 次\n";
        }
    }

    return 0;
}

🔃 do-while 循环

do-while 的特点是先执行一次循环体，再检查条件，因此至少执行一次：

do-while 循环 
#include <iostream>

int main() {
    // 语法：do { 循环体 } while (条件);
    // 注意：while 后面有分号！

    // 基本用法
    int i{1};
    do {
        std::cout << i << " ";
        ++i;
    } while (i <= 5);
    std::cout << "\n";  // 1 2 3 4 5

    // 即使条件一开始就为 false，也至少执行一次
    int x{10};
    do {
        std::cout << "至少执行一次，x = " << x << "\n";
        ++x;
    } while (x < 5);  // 10 < 5 为 false，但循环体已执行

    // 典型应用：输入验证（直到用户输入合法值）
    int age{-1};
    do {
        // 实际程序：std::cin >> age;
        age = 25;  // 模拟用户输入
        if (age <= 0 || age > 150) {
            std::cout << "请输入有效年龄（1-150）\n";
        }
    } while (age <= 0 || age > 150);
    std::cout << "有效年龄：" << age << "\n";

    return 0;
}
循环类型 先检查条件？ 最少执行次数 适用场景
for 是 0 次 已知循环次数，遍历序列
while 是 0 次 不知次数，条件驱动
do-while 否（后检查） 1 次（保证） 必须先执行一次（输入验证）

⏭️ break 与 continue

这两个语句用于控制循环的执行流程：

break 与 continue 对比 
#include <iostream>

int main() {
    // break：立即退出整个循环
    std::cout << "break 演示:\n";
    for (int i = 1; i <= 10; ++i) {
        if (i == 5) {
            break;  // 遇到 5 就退出循环
        }
        std::cout << i << " ";
    }
    std::cout << "\n";  // 1 2 3 4

    // continue：跳过本次循环剩余部分，进入下一次
    std::cout << "continue 演示（跳过偶数）:\n";
    for (int i = 1; i <= 10; ++i) {
        if (i % 2 == 0) {
            continue;  // 跳过偶数
        }
        std::cout << i << " ";  // 只打印奇数
    }
    std::cout << "\n";  // 1 3 5 7 9

    // 在 while 中使用 break 构造无限循环控制
    std::cout << "无限循环 + break:\n";
    int n{1};
    while (true) {  // 无限循环
        std::cout << n << " ";
        if (n >= 5) break;  // 退出条件
        ++n;
    }
    std::cout << "\n";  // 1 2 3 4 5

    // 在 switch 中，break 退出 switch（不是循环）
    // 在嵌套循环中，break 只退出最内层循环
    for (int i = 0; i < 3; ++i) {
        for (int j = 0; j < 3; ++j) {
            if (j == 1) break;  // 只退出内层 for
            std::cout << "(" << i << "," << j << ") ";
        }
    }
    std::cout << "\n";

    return 0;
}

🔲 嵌套循环与乘法表

循环可以嵌套使用，外层循环每执行一次，内层循环会完整执行一遍。嵌套循环是处理二维数据（矩阵、表格）的基本工具：

嵌套循环：打印乘法表 
#include <iostream>
#include <iomanip>   // 用于格式化输出（setw）

int main() {
    // 九九乘法表
    std::cout << "=== 乘法表 ===\n";
    for (int i = 1; i <= 9; ++i) {         // 外层：行（被乘数）
        for (int j = 1; j <= i; ++j) {     // 内层：列（乘数）
            std::cout << j << "×" << i << "="
                      << std::setw(2) << i * j << " ";
        }
        std::cout << "\n";  // 每行结束换行
    }

    // 星号三角形
    std::cout << "\n=== 星号三角 ===\n";
    int rows{5};
    for (int i = 1; i <= rows; ++i) {
        for (int j = 1; j <= i; ++j) {
            std::cout << "* ";
        }
        std::cout << "\n";
    }

    // 矩形方块
    std::cout << "\n=== 方形 ===\n";
    for (int r = 0; r < 4; ++r) {
        for (int c = 0; c < 6; ++c) {
            std::cout << "# ";
        }
        std::cout << "\n";
    }

    return 0;
}
📌

嵌套循环的时间复杂度：如果外层循环执行 n 次，内层循环执行 m 次，那么内层循环体共执行 n×m 次。乘法表的双层循环中，内层执行约 1+2+...+9 = 45 次（不是 81 次，因为内层上限等于外层变量）。

🧪 动手练习

两个经典练习：猜数字游戏（while 循环）和 FizzBuzz 问题（for 循环）：

综合练习 
#include <iostream>

int main() {
    // ===== 练习1：for 循环计算 1-100 的总和 =====
    int sum{0};
    for (int i = 1; i <= 100; ++i) {
        sum += i;
    }
    std::cout << "1 到 100 的总和 = " << sum << "\n";  // 5050

    // ===== 练习2：FizzBuzz（1-20）=====
    std::cout << "\nFizzBuzz (1-20):\n";
    for (int i = 1; i <= 20; ++i) {
        if (i % 15 == 0) {
            std::cout << "FizzBuzz\n";   // 15的倍数
        } else if (i % 3 == 0) {
            std::cout << "Fizz\n";       // 3的倍数
        } else if (i % 5 == 0) {
            std::cout << "Buzz\n";       // 5的倍数
        } else {
            std::cout << i << "\n";      // 其他数字
        }
    }

    return 0;
}

// 预期输出（部分）：

1 到 100 的总和 = 5050

FizzBuzz (1-20):

1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz 11 Fizz 13 14 FizzBuzz 16 17 Fizz 19 Buzz
